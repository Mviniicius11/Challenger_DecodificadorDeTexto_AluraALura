## Text Encoder Project
<br>

Project initially developed as part of an academic course (Oracle + Alura), with the goal of creating a website that allows encrypting and decrypting simple texts.

The current version includes several improvements in responsiveness and adjustments to enhance the visual and user experience.

<br>

### Features:

- **Encryption and Decryption**: The site allows users to quickly encode or decode simple texts.
<br>
<img src="https://github.com/user-attachments/assets/90747d7d-750a-4db3-8045-48b54f7e2313" width="500px" alt="Example image of encrypted text" />
<br>

### How to Use:

1. Enter the text in the input field.
2. Select the desired option (Encryption or Decryption).
3. The processed text will be displayed on the screen.
